1. How to start frontend
- cd trading-website
- npm start

2. how to start backend
- cd flask-server
- python3 -m venv .venv 
- source .venv/bin/activate
- pip install flask flask_cors yfinance
- python3 server.py


or simply do:
- ./run.sh (auto setup for all the front end and back-end code and ran locally)
